### 👨‍💻 Hakkımda
Ben Enes, Sinop Üniversitesi Ayancık Meslek Yüksekokulu'nda **İnternet ve Ağ Teknolojileri 2. Sınıf** öğrencisiyim. Karmaşık iş süreçlerini dijitalleştiren, **Entity Framework Core, ASP.NET MVC** tabanlı kurumsal ölçekli web mimarileri üzerine çalışıyorum. 

Son dönemde özellikle, projelerime **Google Gemini AI** gibi zekaları entegre ederek statik CRUD sistemleri yerine, kendi kararlarını veren proaktif "Otonom Yazılımlar" tasarlamaya odaklandım.

- 🔭 Şu anda geliştirdiğim başyapıt: [**GoldBranch AI - Akıllı Görev ve AI Asistanlık Sistemi**](https://github.com/enesaltndll/GoldBranchAI)
- 🌱 Öğrenmeye / Derinleşmeye devam ettiğim alanlar: **Yazılım Mimarileri, Clean Architecture, AI Destekli Kod Üretimi**
- ⚡ Hobilerim: Yeni teknolojileri beta aşamasında yakalayıp projelerime ilk uygulayanlardan olmak.
