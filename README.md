### 👨‍💻 Hakkımda & Vizyonum
Ben Enes, Sinop Üniversitesi Ayancık Meslek Yüksekokulu'nda **İnternet ve Ağ Teknolojileri 2. Sınıf** öğrencisiyim. Karmaşık iş süreçlerini dijitalleştiren, **Entity Framework Core, ASP.NET MVC** tabanlı kurumsal ölçekli web mimarileri üzerine çalışıyorum. 

Statik CRUD yazılımlar yerine; projelerime **Google Gemini AI** gibi zekaları entegre ederek kendi kararlarını veren proaktif "Otonom Sistemler" tasarlamaya odaklandım.

- 🔭 **Şu anda Geliştirdiğim Sistem:** GoldBranch AI - Akıllı Görev ve C-Level Asistanı
- 🌱 **Derinleştiğim Alanlar:** Clean Architecture, AI Destekli Kod Üretimi, Gelişmiş .NET MVC Yapıları
- 🎯 **Hedefim:** Geleceğin tam otonom kurumsal web ürünlerine imza atmak.
